## 4. 새로 발견된 이슈

### 긴급 (데이터 오류)
1. **부산대학교 grade_type 라벨 오류**: 수시 학생부종합전형 453건이 `admission_type='정시'`, `grade_type='수능등급'`으로 잘못 태깅. 정시 수능등급 쿼리가 수시 결과를 반환하게 됨. `extract_results_batch.py`의 부산대 파싱 로직 점검 필요.

2. **서강대 OCR 쓰레기 학과명 DB 잔존**: "거머르트국저스무", "급도더인국릭부", "경업렉부" 등 OCR 오류 학과명이 DB에 남아있음. API 레벨에서 `_is_valid_dept_name()`으로 필터링되어 외부에 노출은 안 되지만, DB 자체 오염은 지속됨.

### 중간 (기능 불완전)
3. **도전 카드 부재**: 서울 상위권 대학(한양대, 성균관대, 이화여대, 연세대)의 수시 내신 등급 데이터 없어 suggest_portfolio 도전 버킷이 항상 비어있음.

4. **충남대 특성화 전형 필터 누락**: `_is_special_admission("Ⅲ특성화")`가 False 반환하여 exclude_special=True에서 제외 안 됨. 확인 필요.

5. **서강대 전형요소/수능최저 조회 불가**: `admission_process` 테이블 서강대 데이터 없음. Q7 스타일 서류 준비 질문에 답변 불가.

6. **track 필터 오류**: 서강대 중국문화학과가 `track='자연'` 필터 통과. `_matches_track()` 또는 DB track 태그 문제.

### 낮음 (UX 개선)
7. **전형미상 (전형 이름 불명) 데이터 노출**: "(전형미상)" 레이블의 결과가 compare_universities 등에서 상위에 표시됨. 데이터 불확실성 명시 필요.

8. **Q5: search_programs에 university 파라미터 없음**: 특정 학교의 모든 학과를 조회하는 전용 인터페이스 부재.
