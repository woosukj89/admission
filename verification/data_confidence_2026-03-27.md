- **총 건수**: 10,175건

- **총 건수**: 2,413건

- **신뢰도**: 60% — 라벨 오염 문제로 수능등급 쿼리 결과에 수시 데이터 혼입

### 백분위 (정시, score_type='백분위')

### 표준점수 (정시, score_type='표준점수')

- **신뢰도**: 65% — 상대적으로 적은 건수. 일부 환산점수와 혼동 가능성 있음

## 학교별 데이터 품질

### 데이터 양호 학교 (입시결과 + 모집요강 모두 신뢰)
- 경북대학교, 충북대학교, 전남대학교, 제주대학교
- 광운대학교 (fix_kwangwoon 수동 수정 적용됨)
- 국민대학교 (2025 단년도 데이터)
- 인하대학교, 아주대학교

### 데이터 부분 존재 학교
- 서울대학교: admission_process 있으나 입시결과 없음
- 연세대학교: 수능등급 52건 (정시 위주), 수시 내신 없음
- 한양대학교: 표준점수 25건만 보유
- 성균관대학교: grade_type=None 123건 (분류 미완)
- 건국대학교: 환산점수 위주 (58건), 내신 52건

### 데이터 오염/문제 학교
- **서강대학교**: OCR 쓰레기 학과명 다수 (거머르트국저스무, 급도더인국릭부 등), admission_process 없음
- **부산대학교**: grade_type/admission_type 라벨 역전 오류
- **서울과학기술대학교**: grade_type=None 140건 (내신인지 수능인지 불명)
- **성균관대학교**: grade_type=None 123건

### 입시결과 데이터 없는 학교
- 서울대학교 (공식 사이트에서 별도 제공)
- 고려대학교 (서울 캠퍼스) — 입시결과 비공개
- KAIST, POSTECH, UNIST 등 특수대학
- 대부분의 교육대학교

## 수능최저 파싱 현황

| 항목 | 건수 | 비율 |
|------|------|------|
| 수능최저 있음 (전체) | 13,950건 | - |
| 조건 파싱 완료 | 8,708건 | 62.4% |
| 조건 미파싱 (`{있음: True}` 만 존재) | 5,242건 | 37.6% |

**미파싱 원인 분석**:
- 주 원인: extract_admission_batch.py에서 수능최저 원문 텍스트를 attributes에 저장하지 않고 `{있음: True}` 만 기록한 경우. 원문이 없으므로 파서도 분석 불가.
- 가천대학교: 농어촌(교과), 농어촌(종합), 특성화고졸재직자 등 특수전형에서 주로 발생
- 영향: `수능최저_충족` 판별 불가 → AI 모델이 수능최저 충족 여부 못 알려줌

## 사용자에게 반드시 확인 요청해야 할 사항

1. **부산대학교 수시 결과를 정시 쿼리에서 찾을 경우**: 부산대 입시결과에서 `admission_type` 오류. 실제 수시/정시 구분 확인 필요.

2. **서강대학교 정보**: OCR 오류로 학과 데이터 신뢰 불가. 서강대 입학처 직접 확인 권장.

3. **서울 상위권 대학 수시 입결**: 연세대, 한양대, 성균관대, 이화여대 등 수시 내신 데이터 없음. adiga.kr 또는 각 대학 입학처 직접 확인 필요.

4. **수능최저 충족 여부**: 37.6% 전형의 수능최저 조건이 DB에 미파싱. 충족 여부 판별 불가 케이스는 각 대학 모집요강 직접 확인 필요.
