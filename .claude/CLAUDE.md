# 한국 대학 입시 상담 시스템

## 개요
너는 대한민국에 있는 모든 4년제 또 그 이하 대학교들을 대상으로 학생들을 위해 각 학교의 입시 정보를 가져와서 한국에 있는 현 고등학생들 또는 고등학교 교사, 교직원, 상담사 등을 대신하여 상담을 해주는 시스템을 구축하고 있어.

## 기본 기능
1. 전국의 대학교가 출시하는 모집요강 년도 별로 가져오기
2. 전국의 대학교가 출시하는 입시결과 년도 별로 가져오기
3. 그 외의 검증된 데이터 가져오기
4. 모든 데이터를 읽고 쿼리하고 추론하기 쉬운 방법으로 쌓아놓기
5. MCP 형태로 데이터 노출

## 주요 요소
### 데이터의 정확성
모든 추출하는 데이터, 또 저장되는 데이터는 출처가 검증된 데이터여야며 하며, 정확해야 하고, 서로 일관성이 있어야해. 예를 들자면, 같은 대학교의 같은 캠퍼스의 같은 년도의 같은 전형의 같은 모집단위/학과의 같은 전공 사이에 다른 두 데이터가 있으면 안돼. 데이터의 정밀도가 제일 중요해. 이것이 없으면 이 시스템은 무너져.

### 데이터 집계
굉장히 많은 데이터가 다 다른 형태로 존재할거야. 각 대학은 자기만의 전형이름, 학과, 설명 방법, 표시와 표기로 굉장히 다양할거야. 너는 여기에 어떠한 데이터도 잃지 않으면서도 일괄적인 패턴을 이해하고 찾아서 거기에 맞게 데이터를 저장해야 해. 왜나면 이건 나중에 너가 이 데이트를 추출할 때 도움이 될거야.

### 데이터 접근성
데이터가 일관적인 형태로 저장이 되면 너는 API 같은 형태로 이 데이터를 외부에 노출해 줄거야. 다른 추론/생성 AI 들이 이 데이터를 접근해서 학생들을 상담해 줄거야. 이 시스템은 일단은 지금 너의 시스템과는 별개로 생각해 줘.

## 디렉터리 구조
data/
    extracted/ --> 여기는 일차적으로 프로그램이나 AI가 읽기 쉽게 원조 모집요강 데이터를 extract한 결과들이야.
    raw/ --> 이거는 원조 대학별 모집요강 파일들이야
    results/ --> 대학별 년도별 입시결과
    results_extracted/ --> 일차적 입시결과 파일을 extract한 결과값들
docs/ --> 유져가 이해하기 쉽게 문서로 여러가지를 정리해둔 파일들
verification/ --> 데이터를 검증하면서 나오는 모든 레퍼런스 문서 또는 파일들
crawler/ --> 각 대학교 별 웹사이트 또는 그외 검증 사이트들 또한 CDN을 crawling하거나 직접 호출해서 필요한 파일들을 긁어오는 스크립트 또는 helper들
extractors/ --> 각 파일들을 extract 해주는 스크립트들
recommend/ --> 데이터들을 상담을 위해 노출 시켜주는 스크립트들
storage/ --> 데이터 저장소
universities/ --> 전국 대학교들을 가져오기 위한 도우미들
utils/
temp/ --> 일시 파일들, 시스템의 작동을 위해 중요하지 않은 것들. 나중에 정리 필요.

## 검증 방법
데이터를 검증하는 방법은 다양해. 일단 일관성을 확인해야 해. 그리고 너가 알고 있는 최대선에서 보유하고 있는 데이터들이 진실 범위에 있는 지 추측하고 그렇지 않다고 생각하면 verification 안에 문서 형태로 정리해 놔야해. 유저가 직접 확인해야 하는 부분일수록 더 정밀하게 어떤걸 확인해야 하는지 알려줘야 해. 그 외에 누락된 데이터는 없는지, parsing에 문제는 없는지, 이상한 데이터는 없는지, 비정확한 데이터는 없는지 검증하고 너의 검증 상태와 너의 데이터 신뢰도에 대한 최신 평가를 따로 문서에 정리해 줘.

## 테스트 방법
네가 테스트 할 수 있는 방법은 실질적으로 학생들이 할 수 있는 질문들로 하면 돼. 예시를 들자면,

아래와 같은 학생이라고 할때,

| 구분 | 과목 | 등급 |
|------|------|------|
| **내신** (수시 기준) | 국어 | 3등급 |
| | 수학 | 2등급 |
| | 영어 | 3등급 |
| | 한국사 | 2등급 |
| | 과학 | 4등급 |
| | **평균** | **2.8등급** |
| **모의고사** (정시 기준) | 국어 | 3등급 |
| | 수학 | 2등급 |
| | 영어 | 1등급 |
| | 사회 | 4등급 |
| | 과학 | 5등급 |
| | **평균** | **3.0등급** |

1. 모의고사 성적 1학년 국수영사과 32145 등급인데 (예시) 
- 갈 수 있는 학교들 입결(입시성적결과) 높은 순으로 5개 알려줘. 학과 포함해서
2. 내신이랑 모고(모의고사) 비교해서 봤을 때 갈 수 있는 학교 입결 순으로 10개 알려줘. 자연계열로. 무슨 전형인지도
3. 지금 내신이랑 모고 성적으로 인서울 하고싶으면 남은 학기 동안 어디에 집중 하는 게 좋을지 전략 짜줘.
4. 내가 이 내신이랑 모고 성적으로 ㅇㅇ대학교 ㅇㅇ학과 가고싶은데 붙을 확률 알려줘. 또 무슨 성적을 더 높여야 돼? (무슨 과목 또는 내신 등급평균을 남은 학기 동안 몇 등급 이상 계속 유지해야한다는 답변)
5. ㅇㅇ대학교를 가고 싶어. 아무 학과나 상관없이. 가능성 높은 학과랑 전형 순서대로 5개 알려줘. 부족하거나 넘치는 정도도.
6. ㅇㅇ 학과를 가고 싶어. 전국에서 비슷한 학과가 있는 모든 학교들 입결 순으로 나의 합격률 알려줘
7. ㅇㅇ 대학교에 서울 캠퍼스에 ㅇㅇ 전형으로 도전하고 싶어. 내가 지금 준비해야 할 서류들이 뭔지 알려줘.

위에 질문들을 했을때 답변이 뭘지 확인해 보고 - 
- 결과값의 데이터들이 신뢰할 만한지
- 결과값의 데이터들이 정확한 지
- 결과값의 데이터들이 유저가 원하는 것과 일치 하는지
- 결과값의 데이터 중에 누락된 것이 있는지
- 더 보완할 부분이 없는지
체크 해야해.
